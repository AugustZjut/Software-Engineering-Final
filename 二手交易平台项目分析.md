**Root**
- 根目录含 pom.xml（父级聚合 POM，管理版本与依赖）、README.md（产品与功能概览）、second_hand_trading.sql（数据库建表与示例数据脚本）、.gitignore（版本控制忽略规则）、.gitattributes（文本/二进制与 diff 规则）。

**eureka-server**
- pom.xml 继承父级 POM 并引入 spring-cloud-starter-netflix-eureka-server。
- src/main/java/com/second/hand/trading/EurekaServerApplication.java 定义 Eureka 注册中心引导类，端口占用检测后以 8761 启动。
- src/main/resources/application.yml 配置 Eureka 实例主机名、关闭客户端注册/拉取并设置 defaultZone，自定义服务名 eureka-server。

**Server**
- pom.xml Spring Boot 服务 POM，集成 Web、MyBatis、Spring Cloud 依赖，定义编译插件。
- src/main/java/com/second/hand/trading/server/ServerApplication.java 服务入口，启动 Spring Boot 并触发 OrderTaskHandler。
- src/main/java/com/second/hand/trading/server/WebMvcConfig.java 提供全局 CORS 设置与拦截器注册。
- src/main/java/com/second/hand/trading/server/LogCostInterceptor.java HandlerInterceptor 记录请求耗时日志。

Controllers: AddressController（管理收货地址 CRUD 与默认地址），AdminController（管理员登录、列表、状态管理），FavoriteController（收藏新增/校验/列表/删除），FileController（文件上传下载接口），IdleItemController（闲置物品查询、发布、上下架、审核），MessageController（留言新增、列表与回复），OrderAddressController（订单收货地址创建与查询），OrderController（订单创建、状态更新、查询统计），UserController（用户登录注册、资料修改、状态管理、信息查询）。

DAOs: AddressDao（地址数据访问接口），AdminDao（管理员账号表操作与分页），FavoriteDao（收藏增删查验重），IdleItemDao（闲置物查询统计、状态过滤、搜索分页），MessageDao（留言增删改查，按用户或闲置物取列表），OrderAddressDao（订单地址 CRUD 与按订单查），OrderDao（订单 CRUD、分页、统计、按闲置物列表批量查询），UserDao（用户注册登录、状态分类分页、批量查找与信息更新）。

Enum 与异常: ErrorMsg（统一错误枚举常量），ParamException（携带错误字段信息的自定义运行时异常），GlobalExceptionHandler（控制层异常捕获转换为 ResultVo）。

Models: AddressModel（收货地址实体），AdminModel（管理员实体），FavoriteModel（收藏记录实体含嵌套 IdleItem），IdleItemModel（闲置物实体含价格/状态等），MessageModel（留言实体含回复引用、关联用户闲置物），OrderAddressModel（订单配送信息实体），OrderModel（订单实体含支付、金额、关联用户与闲置物），UserModel（用户实体含状态/昵称/头像等）。

Services: AddressService 等接口定义业务操作；其 impl 目录下如 AddressServiceImpl、AdminServiceImpl、FavoriteServiceImpl、FileServiceImpl、IdleItemServiceImpl、MessageServiceImpl、OrderAddressServiceImpl、OrderServiceImpl、UserServiceImpl 实现业务逻辑、参数校验、调用 DAO 及封装返回。

Utils 与任务: IdFactoryUtil（雪花算法或时间戳生成器），OrderTask（定时扫描订单状态任务），OrderTaskHandler（调度器触发定时任务执行）。

VO: PageVo（分页响应封装），ResultVo（统一返回体含状态/消息/数据）。

Resources: application.properties（数据库连接、端口、路径等配置，包括静态资源与上传设定），mapper 目录下各 XML（如 AddressDao.xml 等）编写 MyBatis SQL 映射以及结果集关联。

Tests: ServerApplicationTests.java 基础上下文加载测试。

**WebApp**
- package.json、package-lock.json、babel.config.js、postcss.config.js、vue.config.js、.browserslistrc、.prettierrc、.gitignore 定义前端依赖、构建、格式化与兼容性。
- public/index.html Vue CLI 应用入口模板。

核心源码: src/App.vue 根组件仅渲染 router-view；src/main.js 初始化 Vue、注入 Element UI、全局数据、前置路由守卫；src/router/index.js 定义前端路由、页面组件与元信息。

API 层: src/api/index.js 聚合 axios 请求方法，针对用户、地址、闲置物、收藏、消息、订单、管理员等封装 REST 接口；src/utils/request.js 创建 axios 实例、设置拦截器与基础 URL。

通用组件: src/components/common/AppHeader.vue 顶部导航、搜索、用户信息、登录状态展示；AppPageBody.vue 页面内容容器布局；AppFoot.vue 页脚版权；IdleGoods.vue 后台闲置物管理表格与分页；orderList.vue 后台订单列表与状态操作；userList.vue 用户与管理员管理界面；bus.js 定义事件总线；country-data.js 存放地区数据（文件内容留空或待填充）。

页面组件: components/page/index.vue 首页推荐列表、分类过滤；idle-details.vue 闲置物详情展示与留言；login.vue、sign-in.vue 用户登录注册界面；login-admin.vue 管理员登录页；me.vue 个人中心包含资料与地址管理；message.vue 消息中心；order.vue 用户订单列表；release.vue 发布闲置物表单；search.vue 搜索结果列表；platform-admin.vue 平台后台入口与统计。

**zuul**
- pom.xml 引入 Spring Cloud Netflix Zuul、Eureka client、Spring Boot starter。
- src/main/java/com/second/hand/trading/ServiceZuulApplication.java 启动类启用 ZuulProxy、EurekaClient/DiscoveryClient，端口检测后以 8040 运行。
- src/main/resources/application.yml 配置服务名称、Eureka 注册、路由映射、Hystrix/超时等（需查看具体内容以获取详细路由）。
